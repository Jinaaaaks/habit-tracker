# Habit Tracker Lite

A simple habit tracker built with React + Vite.  
Tracks daily check-ins, streaks, and basic stats. Data persists using LocalStorage.

## Features
- Create habits
- Delete habits
- Daily check-in (one per day)
- Streak tracking and best streak
- Stats summary page
- LocalStorage persistence

## Tech Stack
- React (Vite)
- React Router
- LocalStorage
